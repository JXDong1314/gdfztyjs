## 🛠️ 常见问题排查

### 问题1：部署成功但访问404

**检查项：**
- [ ] 环境 ID 是否正确
- [ ] 是否等待 CDN 缓存刷新（5-10分钟）
- [ ] 在控制台检查文件是否真的上传成功

**解决方法：**
```bash
# 重新部署
cloudbase hosting deploy ./ -e <ENV_ID>

# 清除 CDN 缓存（控制台操作）
```

### 问题2：图片显示不出来

**检查项：**
- [ ] 浏览器控制台是否有404错误
- [ ] 图片路径是否正确
- [ ] 图片文件夹是否都已上传

**解决方法：**
1. 检查文件管理，确认所有文件夹都存在
2. 如果缺失，重新上传：
   ```bash
   cloudbase hosting deploy ./2021年A卷 -e <ENV_ID>
   ```

### 问题3：加载速度慢

**检查项：**
- [ ] 是否开启 CDN 加速
- [ ] 访问地区是否离服务器太远
- [ ] 图片文件是否过大

**优化方法：**
1. 开启 CDN 加速
2. 压缩图片文件
3. 启用 Gzip 压缩（CloudBase 默认开启）

### 问题4：移动端显示异常

**检查项：**
- [ ] 浏览器是否支持 CSS3
- [ ] 是否有缓存问题
- [ ] viewport 设置是否正确

**解决方法：**
- 清除浏览器缓存
- 使用无痕模式测试
- 检查控制台错误信息
